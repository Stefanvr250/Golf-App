description: Create a Product Requirements Document (PRD) for a new feature. Saves to Documentation/[feature]/prd.md.
---

Use the @specify skill to create a PRD from your feature idea. Provide a description of what you want to build, and the skill will guide you through clarifying questions and generate a complete PRD saved in the `Documentation/` folder.

Example: /specify A real-time chat system for customer support with message history and file sharing.
