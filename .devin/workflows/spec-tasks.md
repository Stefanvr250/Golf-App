---
description: Generate an Implementation Plan and Task List from a Technical Specification. Reads Documentation/[feature]/technical-spec.md and writes plan + tasks.
---

Use the @spec-tasks skill to create an implementation plan and executable task list from an existing technical specification. Provide the feature name, and the skill will read the spec and generate the plan and tasks saved in the `Documentation/` folder.

Example: /spec-tasks user-authentication
